Practicas/Proyectos

Aqui te presento mis proyectos mientras estudiaba cursos de HTML, CSS y JavaScript, entre otras tecnologías

Buscador de Autos

El proyecto consta de un arreglo de autos con características, las cuales puedes filtrar para obtener autos con las características en especifico

Validación de Email

El proyecto consiste en validar un formulario que simula el envió de un email.

Administrador de Veterinaria

Este proyecto tiene como fin organizar pacientes de una veterinaria guardando los datos en un arreglo para posteriormente mostrarlos en la interfaz

Adivina el Número

El proyecto es un juego en el cual se genera un numero aleatorio que estará oculto se cuenta con 20 intentos para adivinarlo y cada intento te resta puntuación final

Mapty

Este proyecto sirve para crear puntos de entrenamiento en los cuales ingresas valores para registrar tus tiempos y distancias los cuales se mostraran en la interfaz,es auxiliado por la librería leafletjs que es de código abierto la cual otorga la visualización del mapa.

Bankist V1

El proyecto consta de 4 usuarios albergados en un arreglo con su respectiva contraseña(js:1111, jd:2222, stw:3333, ss:4444) para ingresar, la aplicación simula procesos de un banco como realizar prestamos, transferir dinero o cancelar la cuenta, esta no es la versión final del proyecto puesto que aun lo sigo en curso pero tiene la mayor parte de las acciones ya implementadas.
